# Banprofil mockups

Detta är konceptskisser i SVG-format för hur en första Android-app skulle kunna se ut.

## Grundskisser

- `screen-route-select.svg` - val av rutt
- `screen-train-profile.svg` - inmatning av tåguppgifter
- `screen-main-view.svg` - huvudvy med planvy och profil
- `screen-detail-view.svg` - detaljvy för körning och kommande punkter

## Förfinade skisser

- `screen-route-select-polished.svg`
- `screen-train-profile-polished.svg`
- `screen-main-view-polished.svg`
- `screen-detail-view-polished.svg`

Syftet är att ge ett snabbt visuellt underlag för produktdiskussion. De förfinade varianterna är mer presentabla, men ska fortfarande ses som koncept och inte som slutlig design.
