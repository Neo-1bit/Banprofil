# Banprofil mockups

Detta är enkla konceptskisser i SVG-format för hur en första Android-app skulle kunna se ut.

Filer:

- `screen-route-select.svg` - val av rutt
- `screen-train-profile.svg` - inmatning av tåguppgifter
- `screen-main-view.svg` - huvudvy med planvy och profil
- `screen-detail-view.svg` - detaljvy för körning och kommande punkter

Syftet är att ge ett snabbt visuellt underlag för produktdiskussion, inte att representera färdig design.
